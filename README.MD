# Matty's Halo Tool

![Image of the tool](https://i.imgur.com/tHymOF2.png)

## Some Features:
* Wireframe mode (Universal!)
* Infinite ammo
* No weapon overheat
* Change game speed


This tool is a work in progress, so expect it to be buggy!
